## INTRODUCTION

The Prisoner Hub Protect Approved Content module is a simple module to prevent
editing of specific fields on approved content.

## REQUIREMENTS

This module relies on Content Moderation.
It also relies on a workflow called basic_editorial, and upon specific
fieldnames in use on the Prisoner Content Hub.

## INSTALLATION

Install as you would normally install a contributed Drupal module.
See: https://www.drupal.org/node/895232 for further information.

## CONFIGURATION
No configuration is necessary.

## NOTES
This module protects fields using form validation on node edit forms.
It would not protect any edits produced by other mechanisms, for example bulk
node edits.
